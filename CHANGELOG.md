# Changelog

All notable changes to Versa will be documented in this file.

## [1.0.0] - 2026-06-01

### Added
- 三大模块：资讯 News、辩论 Debate、购物 Shop
- 跨模块融合：
  - 资讯详情页延伸出相关辩论和相关商品
  - 辩论详情页显示来自资讯和相关商品
  - 商品详情页附带媒体评测和相关辩论
- 跨模块声誉系统：8 个等级、9 种勋章、5 维活动追踪
- 完整用户交互：
  - 资讯：阅读进度、3 种反应
  - 辩论：正反方投票、发表观点、热度排序
  - 购物：购物车、收藏、模拟结算
- 个人中心：跨模块足迹、活动流、勋章墙
- 11+ 页面：Home、News（列表/详情）、Debates（列表/详情）、Shop（列表/详情）、Cart、Checkout、CheckoutSuccess、Profile、Wishlist、Orders、About、404
- 暗色 / 浅色 / 跟随系统三主题
- 移动端响应式（底部 Tab 导航）
- 玻璃拟态设计、渐变色徽章、毛玻璃 Header
- GitHub Actions 自动部署到 GitHub Pages
- 完整中文 UI
- TypeScript 类型安全
- React Router 7 + Tailwind 4 + Vite 8 + React 19
